The universal selector * applies styles to every element on the page. Its often used at the beginning of a CSS file to remove default spacing.

Without resetting margins and padding, browsers add their own default spacing to headings, paragraphs, and lists — which can make layouts inconsistent across browsers.

By setting margin: 0 and padding 0 , you ensure all elements start from a clean slate. Then, you can manually add spacing where needed.

Using box-sizing: border-box with the universal selector makes width calculations easier — padding and borders stay inside the elements defined width.

Overall, the universal selector is a simple but powerful tool to keep your CSS consistent and predictable across browsers.
